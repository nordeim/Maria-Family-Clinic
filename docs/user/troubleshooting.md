## Getting Started

### Problem: Cannot create account
**Symptoms:** Registration form not working, error messages during signup

**Solutions:**
1. **Check Internet Connection**
   - Ensure stable internet connection
   - Try different WiFi network or mobile data
   - Restart router/modem if using WiFi

2. **Verify Personal Information**
   - Use exact name as per NRIC/FIN
   - Ensure phone number is Singapore-registered
   - Use valid email address (check spam folder)
   - Age must be 13+ for account creation

3. **Browser/App Issues**
   - Clear browser cache and cookies
   - Disable browser extensions temporarily
   - Try different browser (Chrome, Firefox, Safari)
   - Update browser to latest version

4. **SingPass Registration Issues**
   - Ensure SingPass app is updated
   - Check SingPass service status online
   - Try alternative registration method
   - Contact SingPass support: 6643 9155

**If problems persist:**
- Try registration during off-peak hours (early morning)
- Contact technical support: 1800 TECH HELP (1800 832 4357)
- Use mobile app instead of web browser

### Problem: Cannot verify phone number or email
**Symptoms:** Not receiving verification codes, verification links not working

**Phone Verification Issues:**
1. **Check Phone Number Format**
   - Use 8-digit Singapore number (without +65)
   - Ensure number is registered in Singapore
   - Check for typos in number entry

2. **SMS Issues**
   - Wait up to 5 minutes for SMS delivery
   - Check phone has signal and storage space
   - Verify SMS not blocked by phone carrier
   - Try requesting new code

3. **Carrier Blocking**
   - Contact mobile carrier to unblock SMS
   - Add our SMS number to allowlist
   - Try different mobile network

**Email Verification Issues:**
1. **Check Email Settings**
   - Verify correct email address
   - Check spam/junk/promotions folders
   - Add our domain to safe sender list
   - Check email storage quota

2. **Email Provider Issues**
   - Try different email provider temporarily
   - Check if email provider blocks automated emails
   - Verify email service is working normally

**Alternative Solutions:**
- Use different verification method (phone instead of email)
- Contact customer support for manual verification
- Try registration from mobile app

## Account & Login Issues

### Problem: Cannot log into account
**Symptoms:** "Incorrect username/password" error, account not recognized

**Immediate Solutions:**
1. **Password Reset**
   - Click "Forgot Password" on login page
   - Enter registered email address
   - Check email for reset instructions
   - Create new password following requirements

2. **Username/Email Check**
   - Use email address used for registration
   - Try logging in with phone number instead
   - Check for typos in email address
   - Try different capitalization

3. **Account Status Check**
   - Verify account is not suspended
   - Check email for suspension notifications
   - Ensure account is verified (check email)
   - Try logging in from different device

4. **Browser/Cache Issues**
   - Clear browser cache and cookies
   - Try incognito/private browsing mode
   - Disable browser extensions
   - Try different browser entirely

### Problem: Account locked or suspended
**Symptoms:** Cannot access account, error messages about suspension

**Common Reasons:**
- Multiple failed login attempts
- Suspicious activity detected
- Violation of terms of service
- Extended inactivity
- Payment issues

**Solutions:**
1. **Wait and Retry**
   - Wait 15-30 minutes for automatic unlock
   - Try logging in after waiting period
   - Ensure correct credentials next time

2. **Account Review**
   - Check email for suspension notifications
   - Look for reasons for suspension
   - Follow instructions in notification email
   - Gather required documentation

3. **Contact Support**
   - Call 1800 MY FAMILY (1800 693 2649)
   - Provide NRIC/FIN for identity verification
   - Explain circumstances leading to suspension
   - Follow their guidance for reinstatement

### Problem: Two-factor authentication issues
**Symptoms:** Cannot receive 2FA codes, authentication failing

**SMS 2FA Problems:**
1. **Check Phone Settings**
   - Ensure phone has signal
   - Check phone storage for SMS
   - Verify phone number is correct and updated
   - Try requesting new 2FA code

2. **Network Issues**
   - Try different mobile network
   - Turn phone off and on again
   - Check for carrier SMS blocking
   - Contact mobile carrier if persistent

**App-Based 2FA Problems:**
1. **Authenticator App Issues**
   - Ensure time sync on device is correct
   - Re-scan QR code for setup
   - Try different authenticator app
   - Use backup codes if provided

2. **Alternative 2FA**
   - Switch to SMS 2FA in account settings
   - Use email verification as backup
   - Contact support for 2FA reset
   - Use backup authentication method

## Appointment Booking Problems

### Problem: Cannot book appointment
**Symptoms:** Booking form not submitting, no available time slots, booking errors

**Form Submission Issues:**
1. **Required Fields**
   - Ensure all required fields are completed
   - Check for error messages highlighting missing info
   - Verify NRIC/FIN format is correct
   - Fill in reason for visit

2. **Browser Issues**
   - Clear browser cache and cookies
   - Disable browser extensions
   - Try different browser
   - Update browser to latest version

3. **Network Problems**
   - Check internet connection stability
   - Try booking during off-peak hours
   - Use mobile data instead of WiFi
   - Restart router if connection issues

**No Available Appointments:**
1. **Search Alternatives**
   - Try different dates/times
   - Expand clinic search radius
   - Look for different clinic nearby
   - Try booking earlier/later in the day

2. **Waitlist Options**
   - Join waitlist for preferred times
   - Set up appointment alerts
   - Check for cancellations
   - Try booking 1-2 weeks in advance

3. **Different Booking Methods**
   - Call clinic directly
   - Use HealthHub app
   - Try walk-in for urgent issues
   - Use mobile app version

### Problem: Appointment confirmation not received
**Symptoms:** Booking succeeds but no confirmation SMS/email

**Troubleshooting Steps:**
1. **Check Information**
   - Verify correct phone number and email
   - Check if contact information is updated
   - Look for typos in contact details

2. **Communication Issues**
   - Check spam/junk folders for email
   - Verify phone has signal and storage
   - Contact mobile carrier if SMS blocking

3. **Re-send Confirmation**
   - Log into account and check appointment history
   - Resend confirmation manually if option available
   - Contact clinic directly to confirm appointment

### Problem: Cannot reschedule or cancel appointment
**Symptoms:** Reschedule/cancel buttons not working, error messages

**Common Causes:**
- Appointment too soon for cancellation
- Clinic has specific cancellation policies
- Technical issues with booking system
- Account access problems

**Solutions:**
1. **Time Restrictions**
   - Most clinics require 2 hours notice for cancellation
   - Check clinic's cancellation policy
   - Call clinic directly for last-minute changes
   - Be prepared for potential cancellation fees

2. **Technical Solutions**
   - Log out and log back in
   - Clear browser cache
   - Try different browser or device
   - Use mobile app instead

3. **Alternative Methods**
   - Call clinic directly with confirmation number
   - Use HealthHub app if booked there
   - Visit clinic in person
   - Contact customer support for assistance

## Healthier SG Program Issues

### Problem: Eligibility checker shows "not eligible" incorrectly
**Symptoms:** You meet age/residency requirements but shown as ineligible

**Verification Steps:**
1. **Check Eligibility Criteria**
   - Age: Must be 40 years or older
   - Status: Singapore Citizen or Permanent Resident
   - Location: Must be physically in Singapore

2. **Information Accuracy**
   - Verify age calculation is correct
   - Check NRIC/FIN for typos
   - Ensure contact information matches records
   - Confirm Singapore residency status

3. **System Issues**
   - Try checking eligibility again later
   - Clear browser cache and retry
   - Use mobile app instead of web
   - Contact support if persistent

**Manual Eligibility Verification:**
- Call Healthier SG hotline: 1800 650 6060
- Visit HealthHub to check status
- Contact selected clinic for verification
- Visit MOH service center

### Problem: Cannot find participating doctors or clinics
**Symptoms:** No results in search, limited options in area

**Search Optimization:**
1. **Expand Search**
   - Increase search radius (5km, 10km)
   - Try different postal codes nearby
   - Search by region instead of specific location
   - Check neighboring towns/districts

2. **Filter Adjustments**
   - Remove language filters temporarily
   - Check all clinic types (private and polyclinic)
   - Look for "Accepting New Patients" filter
   - Search for specific doctor names

3. **Alternative Sources**
   - Check MOH official Healthier SG list
   - Call Healthier SG hotline: 1800 650 6060
   - Contact clinics directly to ask about participation
   - Use HealthHub for participating provider search

### Problem: Healthier SG enrollment not going through
**Symptoms:** Stuck in enrollment process, verification failing

**HealthHub Integration Issues:**
1. **SingPass Problems**
   - Ensure SingPass app is updated
   - Check SingPass service status
   - Try different device for SingPass
   - Contact SingPass support: 6643 9155

2. **Data Mismatch**
   - Ensure information matches NRIC exactly
   - Check for name spelling variations
   - Verify address matches official records
   - Update any outdated information

3. **System Overload**
   - Try enrollment during off-peak hours
   - Wait and retry after 1-2 hours
   - Use mobile app instead of web
   - Contact support for manual assistance

### Problem: $20 Healthpoints not received
**Symptoms:** Completed first consultation but no Healthpoints credited

**Healthpoints Issues:**
1. **Timing**
   - Healthpoints may take 3-7 days to appear
   - Check HealthHub app for credits
   - Verify consultation was with Healthier SG doctor
   - Ensure Singapore Citizen status

2. **Verification Steps**
   - Check HealthHub "Wallet" section
   - Look for transaction history
   - Verify doctor marked consultation as completed
   - Check if consultation was within Healthier SG program

3. **Contact Support**
   - Call Healthier SG hotline: 1800 650 6060
   - Provide consultation details and date
   - Include NRIC/FIN for verification
   - Allow 14 days before following up

## Website & App Performance

### Problem: Website loads slowly or times out
**Symptoms:** Pages take long time to load, timeout errors, incomplete page loading

**Network Solutions:**
1. **Check Internet Speed**
   - Test connection speed at speedtest.net
   - Ensure minimum 5 Mbps for smooth operation
   - Consider upgrading internet plan if consistently slow
   - Use ethernet connection instead of WiFi if possible

2. **Network Optimization**
   - Move closer to WiFi router
   - Restart router/modem
   - Try different WiFi network
   - Use mobile data temporarily

3. **Peak Time Issues**
   - Try accessing during off-peak hours
   - Early morning (6-8 AM) typically faster
   - Avoid lunch (12-2 PM) and evening (6-8 PM) peaks
   - Weekend access often faster than weekdays

**Browser Optimization:**
1. **Cache and Storage**
   - Clear browser cache and cookies
   - Disable unnecessary browser extensions
   - Close unused browser tabs
   - Restart browser completely

2. **Browser Updates**
   - Update to latest browser version
   - Try different browser (Chrome, Firefox, Safari)
   - Disable hardware acceleration temporarily
   - Clear DNS cache

### Problem: Features not working on website
**Symptoms:** Buttons don't respond, forms won't submit, pages don't load properly

**JavaScript Issues:**
1. **Enable JavaScript**
   - Ensure JavaScript is enabled in browser
   - Check if JavaScript is blocked by extensions
   - Try different browser to isolate issue

2. **Browser Compatibility**
   - Update browser to latest version
   - Try different browser entirely
   - Check browser developer tools for errors
   - Disable browser-specific features

3. **Security Software**
   - Temporarily disable antivirus/firewall
   - Check if security software blocks website
   - Add our website to trusted sites
   - Use different device to test

**Flash/Plugin Issues:**
- Ensure all required plugins are updated
- Check if flash content is blocked
- Try accessing with plugins disabled
- Update browser plugins to latest versions

### Problem: Error messages during use
**Symptoms:** Various error codes, "Something went wrong" messages, system errors

**Common Error Solutions:**
1. **Error Code Identification**
   - Note exact error message and code
   - Check if error is specific to one feature
   - Try same action again after waiting
   - Document error for support contact

2. **Session Issues**
   - Log out and log back in
   - Clear browser cache and cookies
   - Try different browser or device
   - Check if logged in on multiple devices

3. **Browser-Specific Issues**
   - Try different browser
   - Update current browser
   - Clear all browser data
   - Disable browser extensions temporarily

**System-Wide Issues:**
- Check our status page for known problems
- Try again after waiting 30-60 minutes
- Use mobile app as alternative
- Contact support if issue persists across devices

## Mobile App Issues

### Problem: App won't install or update
**Symptoms:** Installation fails, update not available, download interrupted

**Installation Issues:**
1. **Storage Space**
   - Ensure sufficient device storage (500MB+ free)
   - Delete unnecessary apps or files
   - Clear device cache
   - Restart device and try again

2. **Network Issues**
   - Use stable WiFi for download
   - Pause other downloads/updates
   - Try different network if problems persist
   - Download during off-peak hours

3. **Device Compatibility**
   - Check minimum OS requirements (iOS 13+ or Android 7+)
   - Ensure sufficient RAM (2GB+ recommended)
   - Update device operating system
   - Check available storage space

**Update Problems:**
1. **App Store Issues**
   - Force close and reopen app store
   - Check app store service status
   - Try updating from different device
   - Wait and try again later

2. **Automatic Updates**
   - Disable auto-updates temporarily
   - Manually update when convenient
   - Check app store for available updates
   - Update during charging to avoid interruption

### Problem: App crashes frequently
**Symptoms:** App closes unexpectedly, returns to home screen, won't stay open

**Memory and Performance:**
1. **Device Restart**
   - Force close all apps
   - Restart device completely
   - Clear app cache in device settings
   - Restart device and test again

2. **Storage Issues**
   - Free up device storage (aim for 1GB+ free)
   - Delete unnecessary apps and files
   - Clear app cache and data
   - Move media to cloud storage

3. **App-Specific Solutions**
   - Update app to latest version
   - Reinstall app completely
   - Check for device-specific compatibility issues
   - Contact app support if crashes persist

**Operating System Issues:**
- Update device OS to latest version
- Check device manufacturer for known issues
- Verify sufficient RAM for app operation
- Consider device upgrade if outdated

### Problem: App features not working
**Symptoms:** Buttons don't respond, search not working, bookings fail

**App-Specific Issues:**
1. **App Permissions**
   - Check app permissions in device settings
   - Enable location services if needed
   - Allow camera access for document scanning
   - Ensure notifications are permitted

2. **Network Connectivity**
   - Check app has internet access
   - Try different network (WiFi vs mobile data)
   - Disable VPN if using one
   - Reset network settings if needed

3. **Data Sync Issues**
   - Log out and log back in
   - Force close and reopen app
   - Check sync settings in app
   - Clear app cache and data

**Workaround Solutions:**
- Use web browser as alternative
- Access from different device
- Contact support for feature-specific help
- Use alternate booking methods

### Problem: Offline mode not working
**Symptoms:** App doesn't show cached information, offline features unavailable

**Cache and Storage:**
1. **Cache Management**
   - Enable offline mode in app settings
   - Download required content for offline use
   - Check if data was properly cached
   - Clear cache and re-download if needed

2. **Storage Issues**
   - Ensure sufficient device storage
   - Check if app is allowed offline access
   - Verify app has necessary permissions
   - Restart device and test offline mode

**Feature Limitations:**
- Not all features work offline
- Real-time features require internet
- Check which features support offline mode
- Use offline features appropriately

- Account access completely lost
- Payment processing failures
- Medical emergency while using platform
- Security or privacy concerns

- Time and date when problem occurred

**For Technical Issues:**
- Screenshots of error messages
- Step-by-step description of actions taken
- Browser console errors (if accessible)
