## Accessibility Overview

### Our Commitment to Accessibility

My Family Clinic is committed to providing equal access to healthcare information and services for all users, regardless of their abilities. Our platform meets **WCAG 2.2 Level AA** standards and follows Singapore's accessibility guidelines.

**Key Accessibility Features:**
- Full keyboard navigation support
- Screen reader compatibility
- High contrast and large text options
- Voice control and speech input
- Multilingual support for diverse communities
- Cognitive accessibility features
- Mobile accessibility for all devices

### Understanding WCAG 2.2 AA

**Four Principles of Accessibility:**
1. **Perceivable**: Information presented in ways users can perceive
2. **Operable**: Interface components operable by all users
3. **Understandable**: Information and operation understandable
4. **Robust**: Content works with assistive technologies

**Level AA Compliance Includes:**
- Color contrast ratios of 4.5:1 or higher
- Text resizable to 200% without loss of functionality
- Keyboard accessibility for all features
- Focus indicators clearly visible
- Proper heading structure and navigation
- Alternative text for images and media

## Screen Reader Support

### Compatible Screen Readers

**Windows Screen Readers:**
- **NVDA (NonVisual Desktop Access)**: Free, widely used
- **JAWS (Job Access With Speech)**: Professional screen reader
- **Windows Narrator**: Built-in Windows screen reader

**Mac Screen Readers:**
- **VoiceOver**: Built-in Mac/iOS screen reader
- **Enhanced VoiceOver**: Advanced configuration options

**Mobile Screen Readers:**
- **TalkBack**: Android built-in screen reader
- **VoiceOver**: iOS built-in screen reader

### How to Enable Screen Reader

**Windows (NVDA):**
1. Download and install NVDA from nvaccess.org
2. Press `Ctrl + Alt + N` to start NVDA
3. Use `Insert + F7` to test screen reader navigation
4. Follow voice prompts to navigate

**Mac (VoiceOver):**
1. Press `Cmd + F5` to enable VoiceOver
2. Use `Control + Option + Arrow Keys` to navigate
3. Press `Control + Option + Space` to activate items
4. Use `Control + Option + Command + H` for headings menu

**Android (TalkBack):**
1. Go to Settings > Accessibility > TalkBack
2. Toggle TalkBack ON
3. Use one-finger swipe to navigate
4. Use two-finger tap to scroll

**iOS (VoiceOver):**
1. Go to Settings > Accessibility > VoiceOver
2. Toggle VoiceOver ON
3. Use three-finger swipe to navigate
4. Use double-tap to activate items

### Screen Reader Navigation

**Basic Navigation:**
- **Tab/Shift+Tab**: Move forward/backward through interactive elements
- **Arrow Keys**: Navigate within components (menus, lists)
- **Enter/Space**: Activate buttons and links
- **Escape**: Close dialogs and menus

**Headings Navigation:**
- **H**: Jump to next heading
- **Shift+H**: Jump to previous heading
- **1-6**: Jump to specific heading levels
- **Insert+F7**: List all headings

**Landmarks Navigation:**
- **D**: Jump to next landmark region
- **Shift+D**: Jump to previous landmark
- **Insert+Control+D**: List all landmarks

**Links and Controls:**
- **K**: Jump to next link
- **Shift+K**: Jump to previous link
- **Tab**: Move to next form field
- **F**: Jump to next form field

### Screen Reader Optimized Content

**Form Labels and Instructions:**
- All form fields have proper labels
- Required fields clearly indicated
- Error messages properly associated
- Instructions provided before form fields

**Button and Link Descriptions:**
- Buttons have descriptive text (not just "Click here")
- Links explain their destination or action
- Icons paired with text labels
- Clear action descriptions

**Tables and Data:**
- Proper table headers and associations
- Table captions for context
- Logical reading order
- Data cell relationships clear

**Media Content:**
- Alt text for all images
- Captions for videos
- Audio descriptions when needed
- Transcripts for audio content

## Keyboard Navigation

### Complete Keyboard Accessibility

All platform features are fully accessible using only the keyboard, with no mouse required.

**Basic Navigation Keys:**

**Tab Navigation:**
- `Tab`: Move to next interactive element
- `Shift+Tab`: Move to previous interactive element
- `Enter`: Activate buttons and links
- `Space`: Activate buttons, toggle checkboxes

**Arrow Keys:**
- `↑↓`: Navigate up and down in lists
- `←→`: Navigate left and right in menus
- `Home`: Jump to beginning of list
- `End`: Jump to end of list

**Menu Navigation:**
- `Alt`: Activate menu bar
- `Arrow Keys`: Navigate within menus
- `Enter`: Select menu item
- `Escape`: Close menus

**Dialog Navigation:**
- `Tab`: Navigate between dialog controls
- `Enter`: Confirm dialog actions
- `Escape`: Cancel dialog

### Specialized Navigation Features

**Skip Links:**
- Press `Tab` immediately after page load to see skip navigation links
- "Skip to main content" bypasses navigation menus
- "Skip to search" goes directly to search functionality
- Useful for screen reader and keyboard-only users

**Focus Management:**
- Focus always visible when navigating
- Focus indicators have high contrast
- Focus moves logically through page elements
- Modals trap focus and return to opening element

**Keyboard Shortcuts:**
- `Alt+S`: Jump to search
- `Alt+N`: Jump to navigation menu
- `Alt+L`: Jump to language selector
- `Alt+H`: Go to homepage
- `Alt+M`: Open main menu

### Form Keyboard Navigation

**Field Navigation:**
- `Tab`: Move to next field
- `Shift+Tab`: Move to previous field
- `Enter`: Submit form (unless button has different action)
- `Escape`: Cancel form and close

**Special Field Types:**
- **Dropdown Lists**: Use arrow keys to navigate options
- **Radio Buttons**: Use arrow keys to select
- **Checkboxes**: Use space to toggle
- **Date Pickers**: Arrow keys to navigate calendar

**Auto-complete and Suggestions:**
- Down arrow to open suggestions list
- Up/down arrows to navigate suggestions
- Enter to select suggestion
- Escape to close suggestions

## Visual Accessibility Features

### High Contrast Mode

**System-Level High Contrast:**
- **Windows**: Settings > Ease of Access > High Contrast
- **Mac**: System Preferences > Accessibility > Display
- **Android**: Settings > Accessibility > High Contrast Text
- **iOS**: Settings > Accessibility > Display & Text Size

**Website High Contrast:**
- Automatic detection of system preferences
- Manual high contrast toggle in accessibility settings
- Enhanced color contrast for all text and backgrounds
- High contrast mode for buttons and links

### Text Size and Scaling

**Browser Text Scaling:**
- **Chrome/Edge**: Ctrl + Plus/Minus (Windows) or Cmd + Plus/Minus (Mac)
- **Firefox**: Ctrl + Plus/Minus (Windows) or Cmd + Plus/Minus (Mac)
- **Safari**: Cmd + Plus/Minus

**Website Text Controls:**
- **A-**: Decrease text size
- **A**: Reset to normal size
- **A+**: Increase text size
- Settings saved per browser profile

**Mobile Text Size:**
- **iOS**: Settings > Accessibility > Display & Text Size
- **Android**: Settings > Accessibility > Font Size
- App respects system text size preferences

### Color and Design Accessibility

**Color Contrast Standards:**
- Text: 4.5:1 contrast ratio minimum
- Large text: 3:1 contrast ratio minimum
- Interactive elements: 3:1 contrast ratio minimum
- Focus indicators: 3:1 contrast ratio minimum

**Color-Blind Accessibility:**
- No information conveyed by color alone
- Text labels for all color-coded information
- Patterns and icons supplement color
- Color-blind friendly palette

**Visual Design Features:**
- Clear visual hierarchy
- Adequate white space
- Simple, clean layouts
- Consistent navigation patterns
- Predictable interface behavior

### Zoom and Magnification Support

**Browser Zoom:**
- Zoom up to 200% without loss of functionality
- Horizontal scrolling minimized
- All content remains accessible
- No content cut off or hidden

**Screen Magnification:**
- Compatible with ZoomText and similar tools
- Screen magnifier friendly
- Text remains clear when magnified
- Interactive elements properly sized
